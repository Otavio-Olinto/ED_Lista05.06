# ED_Lista05.06
## Eficiência Bubble, Merge e QuickSort

Este desafio consiste em criar uma aplicação, em Java, que apresente o tempo de ordenação de cada um dos algoritimos para o seguinte vetor:

```
Vetor: {1499, 1498, 1497,..., 3, 2, 1, 0}
```

```
Requisitos do projeto:

1. Arquitetura MVC;
2. O tempo apresentado deve ser em segundos;
3. Cada algoritmo deve ser importado de uma biblioteca indipendente.
```
